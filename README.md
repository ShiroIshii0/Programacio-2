# Programacio-2

## Descripción

Este repositorio contiene los ejercicios de **Programación 2** en diversos lenguajes, principalmente **Java** y **Programación Orientada a Objetos (POO)**. Cada ejercicio tiene su propio laboratorio, con el código fuente, diagramas UML y documentación correspondiente. Los archivos PDF de los laboratorios están organizados en carpetas dentro del directorio `lab_pdfs/`, que incluyen todo el código y los diagramas UML de cada ejercicio.
